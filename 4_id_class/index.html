Id e Class

Id

o id serve como um cpf.Assim como ter um número de cpf, cada elemento só pode ter um valor de id. o cpf é unico, ou seja, ninguem pode ter um cpf igual ao meu. isso significa que o id tambem é único, então não é possivel repetir o mesmo valor duas vezes.

Class

O Class funciona de maneira oposta ai Id. em que voce só pode ter um valor, no Class voce pode ter vários valores. tambem diferentes do id, em que não se pode repetir um valor, no Class no Class voce pode utilizar o mesmo valor para varios elementos
